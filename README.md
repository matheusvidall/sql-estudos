# sql-estudos

Repositório criado para reunir os exercícios e consultas que venho fazendo durante meus estudos de SQL.

A ideia é deixar organizado o que fui aprendendo ao longo do curso, desde consultas básicas até exercícios mais completos com agrupamentos, agregações e joins.

## Estrutura

### `01-select-where-orderby.sql`

Consultas básicas de seleção, filtro e ordenação.

### `02-operadores.sql`

Exercícios com operadores e filtros, como:

* `AND`
* `OR`
* `IN`
* `BETWEEN`
* `LIKE`

### `03-funcoes-agregacao.sql`

Prática com funções de agregação, como:

* `COUNT`
* `AVG`
* `MAX`
* `MIN`
* `SUM`
* `ROUND`

### `04-desafios.sql`

Exercícios um pouco mais completos, misturando filtros, agrupamentos e regras de negócio.

### `05-joins.sql`

Consultas envolvendo relacionamento entre tabelas e prática com joins.

## Objetivo

Usar este repositório como registro do meu aprendizado em SQL e também como base de consulta.
